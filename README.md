# Getting Started
Install the dependencies and run the project
```
npm install
npm start
```

Head over to https://vitejs.dev/ to learn more about configuring vite
## About Basketball Scoreboard

This project is one of the first Javascript solo projects from the Fullstack Developer Path at Scrimba. The board keeps track of the score for the home and guest teams and allows the user to add points in increments of 1, 2, or 3 points.
